# RPG_GAME
RPG躲避小遊戲
簡單的RPG小遊戲，使用JavaFX實現可視化界面，主要使用的技術有:
* Java
* JavaFX

## 功能介紹:
主要有三個關卡，使用者首先選擇一個角色進行闖關並收集金幣，金幣可購買其他角色。
